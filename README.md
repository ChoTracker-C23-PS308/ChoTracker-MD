## Technology Used
- Android Studio with Native Kotlin
- CameraX
- Retrofit
- Gmaps API
- Firebase Authentication
- Kotlin Coroutines

**Note:**
ChoTracker App can only run on Android 5.0 (Android Lollipop) till Android 12 (Android S). The app does not yet support Android 13.
